## Installing Node Using the Node Version Manager (NVM)
According to https://github.com/nvm-sh/nvm , 
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.37.2/install.sh | bash
source ~/.bashrc
nvm list-remote
nvm install v<00000000>
nvm list

```
